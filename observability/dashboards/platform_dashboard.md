# Platform Monitoring Dashboard

Example dashboards include:

## Cluster Health

Metrics:

- Node CPU utilization
- Node memory utilization
- Pod restart count
- Pending pods

## Application Performance

Metrics:

- Request rate
- Error rate
- Latency (p50, p95)

## CI/CD Performance

Metrics:

- Pipeline duration
- Deployment frequency
- Build failures

## Tenant Metrics

Dashboards allow filtering by:

tenant.id
namespace
service.name
